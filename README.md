# React + TypeScript + Vite
用于创建新的 React 18.2.0 项目，开箱即用



## 项目集成

1. React-Router

   同时封装了全局前置路由守卫 `src/router/RouteGuard`

2. Redux

3. Axios

4. Less

5. TailWind

   样式组件

6. antd-mobile

   PC端请卸载 `antd-mobile`，安装 `antd` 即可，同时修改 `src/App.less`




## Use

1. First

   ```bash
   git clone
   ```

2. Second

   ```bash
   yarn install
   ```

3. Third

   ```bash
   yarn dev
   ```

4. Fourth

   ```bash
   yarn build
   ```
